# Announcer for TBC Anniversary

This addon aims to replicate the old WakeSpams addon (credit @cyrila from Bloodfeather-EU).

The goal is to handle announcements of defensive cooldowns, crowd-control, trinket-usage, resists of important spells, and taunts.

Questions should go on the GitHub repo, download page comments, or DM me on discord @ Arviragus#3308.

## Usage

The options can be opened with `/announcer`.

There is likely the ability to toggle on/off anything you might want to toggle on or off, as well as options for how to announce and in what style.
