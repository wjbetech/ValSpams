# Announcer for TBC Anniversary

This addon is my own clone of the old WakeSpams addon designed by a guildie (Cyrila. iirc?) that I played with during original Cataclysm back in the day.

The primary goal is to announce important spell cooldowns (stuns, taunts, CC, so on), item usage (trinkets), and anything else that could be deemed important in a party or raid environment, such as taunts or important spell resists.

WeakAuras is capable of handling some of this functionality, but I wanted to get stuck in with LUA and have everything in one place.
